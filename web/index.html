Установка

Скачивание PWA - это установка из браузера

Откройте SplitApp, добавьте его на главный экран, и он будет запускаться отдельной иконкой без App Store. Все API остаются на том же backend-сервере.

1. Открыть сайт 2. Нажать Install / Add to Home Screen 3. Запустить SplitApp как приложение

Вход

Войти через Яндекс

Откроется страница Яндекса. После подтверждения SplitApp создаст сессию и вернет вас обратно в приложение.
